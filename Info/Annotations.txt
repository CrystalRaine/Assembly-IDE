@Funct, create function, sets up the branches to define a function area, @Funct {name}
@Load, load values from stack, bulk load the given registers from the stack, @Load {reg} {reg} {reg}...
@Store, store values to stack, bulk loads the given registers to the stack, @Store {reg} {reg} {reg}...
@Init, initialize register, bulk initialize registers to the given values, @Init {reg} {const} {reg} {const} {reg} {const}...
@MemoryInit, initialize memory addresses, bulk initializes memory addresses starting at the registers value-address, @MemoryInit {reg} {value} {value} {value}...
